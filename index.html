Rollute Predictor

Welcome to the Rollute Predictor project. This program predicts the next outcome of the Rollute game based on input data using C++.

Files:

  • Download Rollute_Predictor.cpp

Instructions:

  1. Download the Rollute_Predictor.cpp file using the link above.
  2. Compile the file using a C++ compiler (e.g., GCC or Visual Studio).
  3. Run the program and input the last 10 outcomes of the Rollute game.
  4. View the predictions generated by the program.
